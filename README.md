- Your data is saved directly in your browser  
- Information persists between sessions  
- No internet connection is required after the first load  
- No external database or server setup needed

## 🚀 Deploy Online (Free Options)

### **Option 1: Netlify (Recommended)**
- Visit **netlify.com**  
- Drag and drop the HTML file into the dashboard  
- Instantly receive a live URL  

### **Option 2: Firebase Hosting**
1. Create an account at **firebase.google.com**  
2. Install Firebase CLI:  
   ```
   npm install -g firebase-tools
   ```
3. Initialize hosting:  
   ```
   firebase init hosting
   ```
4. Deploy your site:  
   ```
   firebase deploy
   ```
